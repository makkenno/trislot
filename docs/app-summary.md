# アプリケーション概要: Trislot

## 概要
Trislotは、意識的なスキル習得を支援するための個人のためのタスク管理アプリケーションです。
「一度に意識して取り組めることは限られている」という考えに基づき、同時に取り組むスキル（Active Skills）を最大3つまでに制限する「Tri-slot」システムを採用しています。

## 目的
- 新しいスキルや習慣を身につける際、「何を」「いつ」実践するかを明確にする。
- 習得度を可視化し、成長を実感できるようにする。
- 3つに絞ることで、分散しがちな注意力を集中させ、確実な習得を促す。

## 主な機能

### 1. やりたいことリスト (Backlog)
- 今後身につけたいスキルやアイデアを自由に登録・管理できます。
- ここから「習得中」スロットへ移動させることができます。

### 2. 習得中 (Active Skills) - The Tri-slot
- 最大3つまで登録可能な学習中のスキルスロットです。
- 各スキルに対して以下の詳細を設定・記録できます：
  - **実践する場面 (Context)**: 「いつ」「どのような状況で」実践するか
  - **具体的な行動 (Action)**: 「具体的に何をするか」
  - **習得度 (Proficiency)**: 0%〜100%のスライダーで自己評価
- 習得度に応じてランク（入門〜伝説）が表示され、モチベーション維持に貢献します。
- スライダーの目安：
  - 0: 意識してもできない
  - 50: 意識すればできる
  - 100: 無意識でできる

### 3. 習得済み (History)
- 習得が完了した（または100%に達した）スキルを記録・閲覧できます。
- 過去の達成項目として振り返ることができます。

## データ保存
- データはブラウザのIndexedDBに保存され、永続化されます。

## 技術スタック
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Persistence**: IndexedDB
